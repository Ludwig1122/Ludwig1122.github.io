Notes\ C语言main函数参数列表的应用

发表于 2020-11-22 更新于 2020-11-29

前些日子受同事委托，帮忙开发制作一个新的工具程序，在原程序的操作逻辑不变的情况下，通过新增加输入命令即程序名称的参数个数，来增加传入的参数个数，从而满足新的要求；由于生产环境的操作系统是IBM公司的AIX7.1，且编译环境只支持使用Pro C语言，所以这就涉及到了C语言中main函数的参数列表的使用问题。

理解优先：不为空的参数列表

我们知道，在C语言中 main 函数是程序的入口，这里的 main 函数被定义为整型类型，返回值为0；值得注意的是，参数列表括号中没有任何参数的定义，这就意味着在系统中的某处调用这个程序时，除了直接输入程序本身的名字以外，用户不会允许在后面跟进其他任何参数，这样一来可以保证程序成功调用，但是功能和灵活性都非常有限。
